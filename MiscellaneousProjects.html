Class Registration

We hate planning our schedules. There are too many time conflicts! That's why we wrote this program to eliminate the tedious planning process.

Our program assumes days and times for all classes available to engineering freshmen and, depending on whether or not the incoming student has AP credits, selects a subset of courses the student would like to register for. Our algorithm then generates possible schedules for the student that don't have any time conflicts.

Carol and I are both beginning programmers, so we really hope that all our hard work shows in this project. We spent a lot of time working out the algorithm in python while learning python syntax at the same time.

We envision Class Registration at the point where incoming freshmen can text class requests to a phone number run by Twilio, and Twilio will respond with the possible schedules.

Langton's Ant Simulator

Langton's Ant is a cellular automaton, similar to Conway's Game of Life. The "Ant" crawls around a grid, changing squares to different colors, and turns left or right depending on the color of the square it leaves. Langton's Ant is a Turing Machine and can be programes with intial conditions so that it performs computations. It must complete said computations before it reaches "The Highway," when the ant suddently begins to be entirely regular after being seemingly chaotic. On a grid that starts out blank, this occurs after roughly 10,000 steps.

I've programmed Langton's Ant using Java in the Processing IDE. Click below to download the simulation.
